# Cybersecurity Audit Project

## Overview
Comprehensive cybersecurity audit project completed as part of my Masters program at Illinois Institute of Technology (ITMS 578, Fall 2024). This was a group coursework assignment designed to provide hands-on exposure to real-world organizational security assessment practices and enterprise audit methodology.

## Project Scope & Objectives
This audit project involved conducting a thorough organizational security assessment with the following key components:

- **Control Evaluation**: Systematic review and assessment of existing security controls and safeguards
- **Threat Landscape Analysis**: Identification and analysis of potential threats and vulnerabilities facing the organization
- **Vulnerability Identification**: Detection and documentation of security weaknesses and gaps in the current security posture
- **Risk Assessment**: Evaluation of identified risks and their potential impact on the organization
- **Recommendations**: Development of actionable, risk-based recommendations for remediation and improvement
